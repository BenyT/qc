# QC
+ QC是一个前端MVVM框架，适合用来构建复杂的业务逻辑
+ 官网地址:http://time-go.github.io/qc
+ QQ群:330603020

##QC特点:
+ 1.良好的浏览器兼容性(兼容IE8)
+ 2.强大的组件化支持
+ 3.卓越的性能
+ 4.数据智能补齐
+ 5.灵活的自定义事件(内置移动端手势)
+ 6.自动特殊字符转义
+ 7.灵活的动画模块和动画自定义接口

###src文件源码
+ core文件夹里面为核心代码
+ core外面为非核心代码大家可以根据自己需要进行编译 ajax->commonjs->component 依次依赖

###tool文件夹
+ index.html 是一个html代码转字符串数组工具，解决大家拼接字符串的不便

+ 存放示例
